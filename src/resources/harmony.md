## Cadences

| Type | Progression | Effect |
|------|-------------|--------|
| Authentic (Perfect) | V7 → I | Strong resolution; conclusive |
| Plagal | IV → I | Soft; "Amen" cadence |
| Half | any → V | Open; creates tension |
| Deceptive | V → vi | Surprise; avoids resolution |

## Voice Leading Rules

1. **Smooth motion** — move each voice the shortest distance possible
2. **Contrary motion preferred** — when one voice goes up, the other goes down
3. **Avoid parallel 5ths/octaves** — parallel perfect intervals sound hollow
4. **Resolve leading tone** — the 7th scale degree should resolve up to the tonic
5. **Avoid voice crossing** — soprano stays above alto, alto above tenor, tenor above bass
6. **Avoid large leaps** — prefer steps; if leaping, resolve in opposite direction
